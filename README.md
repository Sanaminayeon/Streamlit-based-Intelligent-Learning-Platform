# Streamlit-based Intelligent Learning Platform

# 后端服务器采用Uvicorn结合Fastapi 前端采用streamlit框架

# 首先，请使用Venv创建虚拟环境，并cd到含有requirements.txt的文件夹内，使用pip install -r requirements.txt进行安装依赖

# 请分别打开两个命令行窗口，进入虚拟环境并分别cd至含有登录.py和app.py的文件夹内，其分别位于名为Streamlit-based Intelligent Learning Platform的文件夹与fastapi的文件夹中。

# 执行命令uvicorn app:app --reload来启动后端服务器，执行streamlit run 登录.py启用前端。
